# 雅思听力与阅读练习站

这是一个本地可直接打开的静态练习网站，围绕“背词 + 同义替换 + 打字训练”来设计。

## 当前功能

- `听力模块`
  - 听力词汇：看中文打英文，支持场景筛选、逐词练习、小测模式、错题复习
  - 听力同义替换：支持选择题、输入题、混合模式，现已扩充为剑 10 到剑 20 的完整题库

- `阅读模块`
  - 阅读词汇：第一版基于阅读同义替换核心词整理，按 `Cambridge + Test` 组织题库
  - 阅读同义替换：支持选择题、输入题、混合模式，已接入第一批可稳定整理出的阅读题库

- `打字强化`
  - 单词练习采用“看中文打英文”
  - 点 `看答案` 后，必须自己重新正确输入一遍，才能进入下一题

- `学习记录`
  - 错题本
  - 正确率、答题数、平均用时
  - 听力 / 阅读、词汇 / 同义替换四块数据分开保存到本地 `localStorage`

## 打开方式

直接双击 [index.html](D:/GTD/02_Projects/雅思/听力/词汇练习站/index.html) 即可。

如果你更想用浏览器访问本地服务，也可以在当前目录运行：

```bash
python -m http.server 8000
```

然后打开 `http://localhost:8000`。

## 文件结构

- `index.html`
  页面结构

- `app.js`

- `styles.css`

- `reading-vocab-data.js`

  听力 + 阅读同义替换数据

- `tests/app.test.js`
  Node 测试
