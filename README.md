# IELTS 听力词汇与同义替换练习站

这是一个本地静态练习站，包含两个并列模块：

- `单词练习`
  看中文提示，自己输入英文单词；支持场景筛选、逐词练习、小测模式、错题复习，以及“看答案后必须重新输入正确拼写”。

- `同义替换`
  练雅思听力里常见的同义替换表达；支持 `选择题`、`输入题`、`混合模式`，并拥有独立的错题记录和本地进度。

## 打开方式

最简单的方式是直接双击 `index.html`。如果浏览器限制本地脚本读取，也可以在当前目录下启动一个本地服务：

```bash
python -m http.server 8000
```

然后在浏览器中打开 `http://localhost:8000`。

## 使用说明

- 在左侧先选择要练的模块。
- `单词练习` 可以切换场景、逐词练习/小测模式，以及全部题目/错题复习。
- `同义替换` 可以切换 `选择题`、`输入题`、`混合模式`，并支持错题复习。
- 所有练习数据和进度都会保存在当前浏览器的 `localStorage` 中。

## 文件结构

- `vocab-data.js`：单词题库
- `synonym-data.js`：听力同义替换题库
- `app.js`：页面逻辑
- `styles.css`：页面样式
- `tests/app.test.js`：Node 测试
