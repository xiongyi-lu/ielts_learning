# 雅思听力与阅读练习站

这是一个可以直接本地打开的静态练习网站，核心目标是把 `背词`、`拼写`、`同义替换识别` 放到同一个站点里完成。

## 当前功能

- `听力模块`
  听力词汇：看中文打英文，支持场景筛选、逐词练习、小测模式、错题复习。
  听力同义替换：支持选择题、输入题、混合模式，题库已扩展到剑 10 到剑 20。

- `阅读模块`
  阅读词汇：第一版基于阅读资料整理，按 `Cambridge + Test` 组织练习内容。
  阅读同义替换：支持选择题、输入题、混合模式，已接入可稳定整理出的阅读题库。

- `练习强化`
  单词模式支持 `看答案`，但看完后必须自己重新正确输入，才能进入下一题。
  同义替换支持中文解释，提交后会显示标准答案和释义。

- `练习设置`
  顶部先选择当前主模块：`听力` 或 `阅读`。
  进入听力时，只显示 `听力词汇` 和 `听力同义替换` 两个模式按钮。
  进入阅读时，只显示 `阅读词汇` 和 `阅读同义替换` 两个模式按钮。

- `学习记录`
  错题本
  已答题数
  正确率
  连对状态或当前题型
  听力 / 阅读、词汇 / 同义替换四块数据分开保存到本地 `localStorage`

## 使用方式

直接双击 [index.html](D:/GTD/02_Projects/雅思/听力/词汇练习站/index.html) 即可。

如果你更想用浏览器本地服务打开，也可以在当前目录运行：

```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 文件结构

- `index.html`
  页面骨架

- `app.js`

- `styles.css`
  页面样式

- `vocab-data.js`

  听力 + 阅读同义替换数据

- `tests/app.test.js`
  Node 自动测试
