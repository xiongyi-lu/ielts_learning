# 雅思听力与阅读练习站

这是一个可以直接本地打开的静态练习网站，重点把 `背词`、`英文拼写`、`同义替换训练` 放进同一个练习入口。

## 当前功能

- `听力模块`
  听力词汇：看中文打英文，支持场景筛选、逐词练习、小测模式、错题复习。
  听力同义替换：支持选择题、输入题、混合模式，题库已覆盖剑 10 到剑 20。

- `阅读模块`
  阅读词汇：第一版按 `Cambridge + Test` 组织练习内容。
  阅读同义替换：支持选择题、输入题、混合模式，已接入阅读题库。

- `练习强化`
  单词模式支持 `看答案`，但看完后必须自己重新正确输入，才能进入下一题。
  同义替换支持中文解释，提交后会显示标准答案和释义。

- `复习机制`
  错题会进入学习阶段，并优先回到复习队列。
  连续答对后会逐步延后下次复习时间。
  当前支持 `普通练习`、`今日复习`、`错题专练` 三种练习来源。
  听力/阅读、词汇/同义替换四块进度彼此独立保存。

- `练习设置`
  顶部先选择当前主模块：`听力` 或 `阅读`。
  进入听力时，只显示 `听力词汇` 和 `听力同义替换` 两个模式按钮。
  进入阅读时，只显示 `阅读词汇` 和 `阅读同义替换` 两个模式按钮。

- `学习记录`
  已答题数
  正确率
  错题数
  待复习数
  当前连对或当前题型

## 使用方式

直接双击 [index.html](D:/GTD/02_Projects/雅思/听力/词汇练习站/index.html) 即可。

如果你更想用浏览器本地服务打开，也可以在当前目录运行：

```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 文件结构

- `index.html`
  页面骨架

- `app.js`
  页面逻辑、状态管理、练习流程、复习调度

- `styles.css`
  页面样式

- `vocab-data.js`
  听力词汇数据

- `reading-vocab-data.js`
  阅读词汇数据

- `synonym-data.js`
  听力 + 阅读同义替换数据

- `tests/app.test.js`
  Node 自动测试
